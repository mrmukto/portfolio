# CV Website (GitHub Pages)

This folder is meant to be published with **GitHub Pages**.

## Quick Setup

- Put your profile photo at `docs/assets/profile.png`.
- Enable GitHub Pages:
  - Repo **Settings** → **Pages**
  - Source: **Deploy from a branch**
  - Branch: `main`
  - Folder: `/docs`

Your site will be available at:

`https://<username>.github.io/<repo>/`
